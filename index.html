Setup Firebase (Free)
Google's free database — no credit card needed, free forever.
Step 1: Go to console.firebase.google.com
Step 2: Click "Add project" → name it "a2dv-fund" → Continue
Step 3: Disable Analytics → "Create project"
Step 4: Left sidebar → Build → "Firestore Database"
Step 5: Click "Create database" → "Start in test mode" → Enable
Step 6: Gear icon → Project Settings → click "</> Web" → register app → copy the firebaseConfig
