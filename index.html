Setup Firebase 🔥
Google's free realtime database — takes 5 mins, no credit card needed.
1. Go to console.firebase.google.com
2. Add project → name it "a2dv-fund" → Continue
3. Disable Analytics → Create project
4. Sidebar → Build → Firestore Database
5. Create database → Start in test mode → Enable
6. Gear ⚙️ → Project Settings → click </> Web → register → copy firebaseConfig
